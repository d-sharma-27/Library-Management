# Library Book Management System

## Project Title
Library Book Management System

## Overview
A simple Python console application to manage library books. Features include adding books, borrowing, returning, and listing books with available and total copies.

## Features
- Add new books or increase copies of existing books
- Borrow books (availability decreases automatically)
- Return books (availability increases)
- List all books with author and stock details
- Beginner-friendly modular code (book_manager.py + main.py)

## Technologies / Tools Used
- Python 3.x (no external libraries required)

## Steps to Install & Run
1. Ensure Python 3 is installed.
2. Clone the repository or download the project files.
3. Open terminal / command prompt and navigate to project folder.

## Instructions for Testing
- Start the program and test the following commands:
  - `add` → Follow prompts to add a book.
  - `borrow` → Enter title to borrow (if available).
  - `return` → Enter title to return.
  - `list` → Shows all books.
  - `quit` → Exit the program.
